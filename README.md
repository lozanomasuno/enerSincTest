## Run Locally

Clone the project

```bash
  git clone https://github.com/lozanomasuno/enerSincTest.git
```

Go to the project directory

```bash
  cd enerSincTest
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start 
```
```bash
  yarn  start 
```
